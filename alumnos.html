Sistema Alumnos


Previous Next

El proyecto es una pequeña API-Rest que simula un sistema de inscripción para una Academia de Música.
El proyecto consta de tres partes:
La primera parte es el diseño de una base de datos relacional con información acerca de los cursos y alumnos.
La segunda parte es un proyecto Java utilizando el framework Springboot para el diseño de una API-Rest bajo el modelo MVC.
El proyecto implementa un CRUD con Jpa Repository, para todas las entidades.
La tercera parte es una aplicación de consumo de la API, despliega la información de los alumnos y los cursos disponibles, se puede agregar materias y se pueden inscribir alumnos.
Se desarrolla bajo el patrón de diseño MVC.

Tecnologías :
Java8/API-Rest/MySql/SpringBoot/HTML/Thymeleaf/Bootstrap/BackEnd/FrontEnd
