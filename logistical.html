Empresa Logistical


Previous Next

El proyecto simula una aplicación de control de registros de una empresa de venta de zapatillas.
Se desarrolla para aplicar conceptos relacionales dentro de un proyecto Java SpringBoot.
La aplicación permite desplegar la información de las existencias y paginar la información desplegada.
Se puede agregar, modificar y eliminar registros.
El proyecto implementa un CRUD con Jpa Repository, para todas las entidades, además de algunas querys nativas específicas.
Se desarrolla bajo el patrón de diseño MVC.

Tecnologías :
Java8/MySql/SpringBoot/Jpa/Jsp/CSS/Bootstrap/
